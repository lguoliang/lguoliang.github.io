# 对象和数组的扩展用法

… 扩展运算符能将 数组 转换为逗号分隔的 参数序列

```js
const args = ['1','2','3'];
change function(){
    console.log(arguments)
}
change(...args);//change('1','2','3');
change(args);//change(['1','2','3'])

```

应用：
1. 数组的合并
  ```js
  const str = […str1,…str2];
  ```

2. 数组的克隆
  ```js
  const str = [‘1’,‘2’,‘3’];
  const str1 = […str]; //[‘1’,‘2’,‘3’]
  ```

3. 将伪数组转为真正的数组
